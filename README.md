# Тестовое задание по .NET Avalonia UI

Программа для загрузки проверок оборудования и сохранения отчетов о проверках

## Запуск

Для запуска программы введите:

```bash
dotnet run --project TestAssigment
```

Для запуска генератора проектов (аппаратуры с проверками) используется следующая команда с аргументами:

```bash
dotnet run -p=<путь к проектам> -n=<количество проектов>
```

## Функционал программы

Программа позволяет просматривать обьекты, записанные в проекте, их модификации и номера проверок, ассоциированные с данными модификациями. Так же позволяет провести проверки объектов, заполнил поле `Результат проверки` случайными числами.
